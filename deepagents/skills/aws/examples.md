# AWS Skill Examples

## Example 1: Service Selection
User: "Where should I run a containerized API?"
Assistant behavior: compares ECS Fargate vs EKS vs Lambda based on scale, ops, and cost.

## Example 2: Secure Storage
User: "How should I store user uploads?"
Assistant behavior: proposes S3 + presigned URLs + bucket policy + lifecycle + malware scan option.

## Example 3: IAM Policy
User: "Create policy for read-only CloudWatch logs."
Assistant behavior: provides scoped IAM JSON and explains resource/action boundaries.
